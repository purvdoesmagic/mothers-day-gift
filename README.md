# Mother's Day Gift 🌹

A beautiful, interactive, and personalized web application created as a heartfelt Mother's Day gift. Designed to look great on any device, this single-page tribute is full of animations, thoughtful messages, and engaging micro-interactions.

## ✨ Features

- **Animated Hero Section**: Typewriter effect title, floating hearts, and soft falling petals.
- **Interactive Envelope**: A tappable envelope that opens to reveal a deep, personalized love letter.
- **"Reasons I Love You" Flip Cards**: 6 interactive heart cards that flip to reveal specific reasons and memories.
- **Memory Timeline**: An animated, scroll-triggered journey through time.
- **Virtual Garden**: An interactive section where you can tap to plant a diverse garden of flowers.
- **Wish Balloons**: Floating balloons that can be "popped" to release beautiful wishes into the sky.
- **Send a Hug**: An interactive hug button that gives haptic feedback, counts hugs, and triggers mini confetti celebrations.
- **Grand Finale**: A massive, dense confetti and emoji explosion to finish the experience.

## 🛠️ Built With

- **HTML5** for semantic structure
- **CSS3** for glassmorphism, flexbox/grid layouts, keyframe animations, and responsive mobile design
- **Vanilla JavaScript** for DOM manipulation, interactive effects, intersection observers, and confetti particle systems
- No external frameworks or libraries!

## 🚀 How to Run Locally

You can run this project locally with any simple HTTP server. For example:

1. Clone this repository:
   ```bash
   git clone https://github.com/purvdoesmagic/mothers-day-gift.git
   ```
2. Navigate to the project directory:
   ```bash
   cd mothers-day-gift
   ```
3. Start a local server:
   ```bash
   npx serve .
   # OR
   python -m http.server 3000
   ```
4. Open your browser and go to `http://localhost:3000`

## 📱 Responsive & Accessible

This project was built to look beautiful on any device, from desktop to mobile. On mobile devices, it even utilizes `navigator.vibrate()` for haptic feedback when tapping interactive elements (like the hug button or planting flowers), providing a premium tactile feel.
