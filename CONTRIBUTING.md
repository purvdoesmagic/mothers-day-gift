# Contributing to Mother's Day Gift 🌹

First off, thank you for considering contributing to this project! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

Since this project was originally built as a personal Mother's Day gift, the core structure is complete. However, if you have ideas on how to make the animations smoother, the layout more responsive, or want to add cool new features that others could use for *their* moms, your suggestions are highly welcome!

## How to Suggest a Change

If you want to suggest a change, the best way to do it is by submitting a **Pull Request (PR)**. Here is how you do that:

### 1. Fork the Repository
Click the **Fork** button in the top right corner of this page. This will create a copy of the repository in your own GitHub account.

### 2. Clone Your Fork
Clone the copied repository to your local machine:
```bash
git clone https://github.com/YOUR-USERNAME/mothers-day-gift.git
```

### 3. Create a Branch
Create a new branch for your feature or bug fix:
```bash
git checkout -b cool-new-feature
```

### 4. Make Your Changes
Open the code in your favorite editor, make your changes, and test them locally by opening `index.html` in your browser (or running a local server).

### 5. Commit and Push
Once you are happy with your changes, commit them and push them up to your fork on GitHub:
```bash
git add .
git commit -m "Add a cool new feature"
git push origin cool-new-feature
```

### 6. Open a Pull Request
Go back to the original `mothers-day-gift` repository on GitHub. You should see a prompt to **Compare & pull request**. Click it, write a brief description of what you changed and why, and submit it!

I will review your suggested changes, and if everything looks good, I'll merge them into the main project. 

## Issues
If you don't want to write code but found a bug or have a feature request, feel free to open an **Issue** on the GitHub page describing what you'd like to see changed.

Thanks again for helping make this project better! 💖
